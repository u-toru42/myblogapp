# アプリケーションの概要
画像投稿機能付きのブログです。

# アプリケーションの機能一覧
- 記事投稿機能
- 画像投稿機能
- 投稿一覧機能
- 投稿詳細機能

# アプリケーション内で使用している技術一覧
- 言語 Python
- フレームワーク Django
- インフラ AWS(Amazon EC2にUbuntu/PostgreSQL/Nginx)を設定し公開
- データベース PostgreSQL
